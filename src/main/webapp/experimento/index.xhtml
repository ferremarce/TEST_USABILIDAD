Bienvenido

Gracias por participar en este experimento donde vas a evaluar la usabilidad de una aplicación Web.

Esta prueba la puedes hacer una sola vez. La prueba consta de 3 tareas que se harán sobre una Aplicación Web que gestiona una tienda virtual.

Durante toda la prueba no debes hacer click en el botón "RETROCEDER O ATRÁS" del navegador Web donde estás visualizando esta página.
El tiempo estimado para hacer las tareas es de 7-12 minutos.

Una vez empieces la prueba la debes terminar sin interrupciones. No te debes distraer con cualquier otra cosa, como por ejemplo el messenger, whatsapp, skype, etc.

Se te va a controlar el tiempo que tardas en finalizar las tareas y si inviertes tiempo en cosas distintas a la prueba, los tiempos no serán realistas. Por tanto, es recomendable que apagues el messenger, skype u otro mecanismo de mensajería.

Al final de todas las tareas deberás contestar a un cuestionario en base a la experiencia obtenida en el uso del sistema.

A continuación, tienes un cuestionario para recoger tus datos de interés. Una vez que lo finalices, pulsa sobre el botón "SIGUIENTE" para iniciar el entorno del experimento.
