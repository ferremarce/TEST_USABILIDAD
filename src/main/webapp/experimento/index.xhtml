Bienvenido

Gracias por participar en esta prueba donde vas a evaluar ciertos aspectos de una aplicación Web.

Esta prueba la puedes hacer una sola vez y consta de 3-4 tareas que se harán sobre una aplicación web que gestiona una tienda virtual.

Durante toda la prueba no debes hacer click en el botón "RETROCEDER O ATRÁS" del navegador web donde estás visualizando esta página.

El tiempo estimado para hacer las tareas es de 7-12 minutos.

Una vez empieces la prueba la debes terminar sin interrupciones. No debes distraerte con cualquier otra cosa, como por ejemplo el messenger, whatsapp, skype, etc.

Se te va a controlar el tiempo que tardas en finalizar las tareas y si inviertes tiempo en cosas distintas a la prueba, los tiempos no serán realistas. Por tanto, es recomendable que apagues el messenger, skype u otro mecanismo de mensajería.

Al final de cada tarea deberás contestar un breve cuestionario en base al uso que le has dado al sistema.

A continuación, tienes un formulario inicial para recoger tus datos de interés, donde debes ingresar un Nombre o Alias necesario para la evaluación.
